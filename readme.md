Shiny ID3 Tagger
================
A straight-forward windows application to automatically retrieve missing ID3 tags for your music files

Currently supported webservices
- 7digital
- Amazon
- Quantonemusic (formerly Decibel)
- Deezer
- Discogs
- Genius
- Gracenote
- iTunes
- Last.fm
- Microsoft Groove
- Musicbrainz
- Musicgraph
- Musixmatch
- Napster
- Qobuz
- Spotify
